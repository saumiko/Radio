
 LAMEOnJ

 Java API based on LAME MP3 encoder/decoder

---------------------

* build_LAMEOnJ.xml 

 Is an Ant file IDE agnostic.

* build.xml 

 Is generated by NetBeans, in fact LAMEOnJ project is a NetBeans project. Individual files can be compiled using NetBeans menues/shortcuts but use build_LAMEOnJ.xml for any other development task (deploying etc).
